# Yazgi
Yazgınız, kaderiniz sizin elinizde. Yazgı, klasik metin maceralarından ilham alan; ancak statik senaryolar yerine yapay zeka ajanlarıyla dinamik olarak örülen hikayeler sunan açık kaynaklı bir oyun motorudur. Her komutunuz dünyayı kalıcı olarak etkiler, her oynanış farklı bir yazgı doğurur.
